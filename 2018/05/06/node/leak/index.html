二、定位的方法

2.1、node inspector

V8现在已经可以直接 inspector 了, 只需要执行 node --inspect ***.js 就可以配合chrome查看内存信息

具体步骤

  1. node --inspect ***.js
  2. 打开chrome://inspect/#devices
  3. 打开开发者工具 profile
  4. 点击 Take SnapShot

这样就可以根据右侧的信息，查看哪个对象暂用的内存过大
